## 时间序列

- 生成时间序列index

  ```python
  import pandas as pd
  time_index = pd.date_range(start='2019-06-20 06:00:00', 
                             end='2019-06-22 15:00:00', freq='H')# freq参数可以指定生成序列的时间间隔'H'每小时，'D'每天等等
  print(time_index)
  ```
